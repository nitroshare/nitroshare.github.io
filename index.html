We're Moving Our Code!

Our code is being moved from LaunchPad to GitHub. Instead of a single project that builds all of the shared libraries and the GUI client, we will soon have separate projects - each with their own version numbers.

Components

The entire NitroShare project consists of the following components:


libnitroshareutil

Provides a number of utility classes that are used by the other shared libraries

libnitroshareproto

Provides an implementation of the NitroShare protocol (NSP)

libnitrosharegui

Provides a number of graphical components used by the desktop client

nitroshare-client

The graphical desktop client that runs on client devices
