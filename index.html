Our code is being moved from Launchpad to GitHub. Instead of a single project that builds all of the shared libraries and the GUI client, we will soon have separate projects - each with their own version numbers.

Components

The entire NitroShare project consists of the following components:


QHttpEngine

Provides a simple set of classes for developing HTTP server applications in Qt.

QDiscovery

Simplifies the process of developing network applications that discover other peers on a local network and exchange data.

nitroshare Daemon

The CLI client that runs on client devices.
